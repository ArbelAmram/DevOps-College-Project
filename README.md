# DevOps-Project
Simple code integration between several platforms

# Project Idea
The main focus of this project is to be able to integrate between several platforms
in order to simulate a product development process (DevOps).

# Coding File
The coding file (JSP) is a basic web Calculator.
supporting Addition, Multiplication and Division.
index: first window - input of values and selecting function.
a: the second window - displaying result and an additional button which enable to returning to the first window.

# Tools & Platforms:
Git - *Push* the coding file to git account.
Jenkins - chaining processes.
Apache Tomcat - channeling the JSP file to localhost
Selenium - Automate the functionality.
Ngrok - receiving secure URL to the localhost server
SiteMonitor - Upload the JSP to a remote server and check running (online - "onAir") status.
LoadRunner Cloud - load testing through creating multiple HTTP requests to the website.
TruClient - testing tool for performance, load, automation and monitoring.
